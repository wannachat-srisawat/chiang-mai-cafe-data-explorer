# Chiang Mai Cafe Data Explorer

Interactive cafe analytics platform built with React, TypeScript, Leaflet, Recharts, SWR, and Supabase.

## Features

### Cafe Management
- Create / Edit / Delete cafes
- Image upload
- Search and filtering
- Pagination

### Map Experience
- Interactive map
- Marker clustering
- Heatmap
- District filtering

### Analytics Dashboard
- Cafes by district
- Price distribution
- Feature distribution
- Top vibes

### Location Intelligence
- Opportunity score
- District comparison
- Recommendation engine

## Tech Stack

- React
- TypeScript
- SWR
- Tailwind CSS
- Leaflet
- Recharts
- Supabase
